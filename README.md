House_Price_Crawler
===
Crawl house price by scrapy.

Target site
---
```
https://community.houseprice.tw/list/%E5%8F%B0%E5%8D%97%E5%B8%82_city/%E6%9D%B1%E5%8D%80_zip/
```

Run code
---
Just run
```
scrapy crawl HousePrice
```

Output csv file
```
scrapy crawl HousePrice -O house_price.csv
```
